HTML

HTML (Hypertext Markup Language) is all about the structure of the webpage, features and elements that we need to add for our proffesional website

Before structuring a webpage using html, we have to learn about the HTML tags that helps the browser to understand what we should be added for our website

Elements & HTML Tags

An element is a segment of HTML that visibles specific components of a web page like images,texts,buttons,forms,tables,links etc. HTML tags contains the elements and attributes that structures your webpage.They are enclosed with brackets (<>).Most of the tags consists of an opening tag<> and a closing tag<>. The tags that has a closing tag are called container tag and the tags that consist of only opening tag are called empty tags

Some of the most important element tags in html are:-

Container tags

CSS

CSS(Cascading Style Sheets) is where we get to style our web page.CSS is what makes our webpage more presentable to our preffered liking. Throught CSS, we are able to style the texts,images,shapes etc.

Before we learn how to style in css, we need to know about some of the properties of css

Some of the most important CSS properties are:-

  • Font size property(font-size):-This property can be used to change the sixe of a text or heading.It uses the value of pixels (px) and percent(%).
    •            eg:-
           font-size:20px
  • Background color property(background-color):-This property helps in changing the color of the background.
    •            eg:-
           background-color: gray
  • Color Property(color):-Color property changes the colour of text,lines etc.
    •            eg:-
           color: gray
  • Text Align Property(text-align):-This property enables us to align the texts ans paragraphs in our preffered liking.
    •             eg:-
            text-align:center
  • Width Property(width):-This property enables us to change the width and size of a shape or an image to our preference.
    •                 eg:-
                width:40px
  • Height Property(height):-This property enables us to change the height and sixe of a shape or an image to our preference.
    •                 eg:-
                height:40px
  • Font Family Property(font-family):-Font family Property enables us to select a specific font according to our preference for our text ,pragraphs and headings.
    •                 eg:-
                font-family:- Arial, Helvetica, sans-serif
  • Border Tag:- This tag can increase or decrease the thickness of the border of the shape.
    •             eg:-
            border:2px solid
            Here,the solid symbolises the style of the border.

These are some of the most important Elements and Properties of HTML & CSS before building an HTML webpage.
